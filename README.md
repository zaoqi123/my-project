# my-python-demo 🚀

![Python版本](https://img.shields.io/badge/Python-3.8%2B-blue)

这是 **zaoqi123** 的第一个Python项目，演示如何发布到GitHub！

## 功能
- 打印个性化欢迎语
- 兼容Python 3.8+

## 快速开始
1. 克隆仓库：
   ```bash
   git clone https://github.com/zaoqi123/my-python-demo.git
   ```
2. 安装依赖（可选）：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行程序：
   ```bash
   python main.py
   ```
